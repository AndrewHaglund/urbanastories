# About This Site

*House Stories* is produced by Nicole Haglund. This website acts as the digital accompaniment to the physical signs mounted near the mentioned homes.

The website is built using [Eleventy](https://www.11ty.dev/), hosted by [GitHub pages](https://pages.github.com/), and developed by [Andrew Haglund](https://haglund.app/).

The [code](https://github.com/AndrewHaglund/urbanastories) for this project is open source.
